# 大事件  

:::tip 提示

在这里记录一些重大的新功能或重大变更。  

:::

## 静态版本（2023-01-14）  

从`1.18`版本开始，提供了一个简化版本的纯前端大屏。  
不需要任何的权限，数据也完全保存在用户浏览器的缓存当中。  
支持配置导出导入以及初始化。  
允许预览。  
[静态版本地址](https://food-billboard.github.io/create-chart/index.html)    

## 移动端支持（2022-11-18）  

从`1.15`版本开始，大屏支持**移动端**的支持。  
现在可以在设计器中设计移动端的大屏了！！  
并且也提供了电脑端转移动端的支持！！
