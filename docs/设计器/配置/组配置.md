# 组  

多个组件的*集合*，**组**内的组件的*位置*和*大小*均和**组**相关。  

很多情况下，一个大屏中总会存在很多类似的模块，比如**标题**+**图标**的组合。  
完成一个组合的制作时，通过将其**成组**，然后**复制粘贴**，可大大减少操作步骤，并且**成组**后更加易于*管理*。 

:::tip

修改**组**的尺寸时，**组**内的组件的尺寸不会发生变化，但是会发生**缩放**。

:::
