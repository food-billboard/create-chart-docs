模板列表

介绍​

对大屏的模板进行统一管理，模板功能帮助用户预先配置一些包含通用样式或功能或数据的大屏。

可以在其之上进行二次的修改，制作，加快大屏成型速度。

基本的功能与大屏的类似。

功能​

使用模板​

跳转至设计器页面，并且已经预先将模板的配置合并到大屏的数据中作为默认数据。

其他功能​

其余功能包括 预览、启用禁用、删除、导入导出。具体功能可查看大屏文档。传送门
