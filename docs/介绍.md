# 可视化大屏操作文档

**自己做的一个可视化数据大屏**。

## 介绍

这是一个自己做的可视化数据大屏，只是自己简单娱乐的工具。当中包含了可视化大屏的大部分基础功能，一些高级功能还待研发。

:::tip 提示

可视化大屏当中一些包含复杂后台交互的功能不在本人的研发计划内  

:::

如果有读者有兴趣或者想二次研发的话，可以去仓库查看源码。

**[仓库地址](https://github.com/food-billboard/create-chart)**

### 说在前头  

在真正使用到此大屏时，需要先注册账号。

**在此声明**  

本人不会对使用此大屏的账号信息做任何的功用，只为了保证后台数据的清楚规整。   
如果介意可以不使用，谢谢🙏🏻。
