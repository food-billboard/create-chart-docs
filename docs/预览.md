# 预览  

## 介绍  
  设计制作大屏时，有一部分的效果需要在最后查看的时候才能看到。  

  比如 一些图表📈组件有`tooltip`数据提示框，设计器不响应一些*鼠标的动作*，所以可以在**预览**的模式预先查看效果。  
  又比如 **列表**组件的滚动动画，同样可以在**预览**模式查看。  

:::note 提示

预览不同于分享，只有**登录**的状态下才能*预览*。  

:::
